Web Speech API SpeechSynthesis interface: hints demo

This demo uses the SpeechSynthesis interface of the Web Speech API. Check CanIUse.com for current browser support.

Example

When keyboard or mouse focus is placed on the button, a hint is spoken by the default platform Text To Speech (TTS) voice.

Notes

This technique could be used to provide help for people with cognitive disabilities, or people unfamiliar with the web. It would not be needed for simple controls (this is just a demo), but it could be used for complex controls like sliders, tabs etc.

If help was made available in this way, it would need to be made "opt-in". Most people will not want the hints to be spoken.
