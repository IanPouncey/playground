CSS Speech demo

The demo on this page uses the SpeechSynthesis interface of the Web Speech API, to simulate support for the CSS Speech module. Check CanIUse.com for current browser support.

Example

Choose a Text To Speech (TTS) voice, then use the tab key to move through the content.

Notes

This demo demonstrates how a screen reader might respond to properties from the CSS Speech module. Using the Web Speech API to simulate CSS Speech support has more information.
